Tech Glossary

The terms your engineers use that no one explained to you — until now.

Term Plain English Context
dataLayer A JavaScript array that acts as a communication bridge between your site and GTM. GTM
crawl budget How many pages Googlebot will crawl in a given timeframe — finite and precious. SEO
JOIN Merging two SQL tables on a shared column. The core of relational databases. SQL
HTTP 301 Permanent redirect. Passes ~90% of link equity. Use for canonical URL migration. SEO / Web
event parameter Extra data attached to a GA4 event (like product_id, page_type, or scroll_depth). GA4
REST API A standard way to request data from a server using URLs and HTTP methods (GET, POST). APIs
